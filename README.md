# Student Result Analyzer

This project is a Python-based console application that analyzes student academic performance. It calculates total marks, averages, assigns grades, and identifies top-performing students.

## Features

* Add student details and marks
* Calculate total and average marks
* Assign grades based on performance
* Identify the topper
* Display all student results
* Includes sample data for testing

## Technologies Used

* Python
* Lists and Dictionaries
* Basic Input/Output Handling
